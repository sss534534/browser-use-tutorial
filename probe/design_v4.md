## 方案迭代 v3 — 向生产级别演进                                                                                                                                                                        
                                                                                                                                                                                                            
     ---                                                                                                                                                                                                    
                                                                                                                                                                                                            
     ### 一、v2 遗留的未解问题                                                                                                                                                                              
                                                                                                                                                                                                            
     | 问题 | 根因 | v3 必须解决 |                                                                                                                                                                          
     |------|------|------------|                                                                                                                                                                           
     | **分布式并行是假的** | delegate_task 的多 agent 是独立进程，无法共享一个 browser session | 引入 Selenium Grid / Playwright Cluster 做真并行 |                                                        
     | **自愈停在"注册表更新"** | 策略更新后没有验证、没有版本控制、无法回滚 | 自愈必须有验证闭环 |                                                                                                         
     | **LLM 动态生成 locator 从未实现** | 只提了方向，没有给出调用链路 | 必须给出具体 prompt + 调用流程 |                                                                                                  
     | **视觉回归测试是空白** | 只用 vision 做兜底定位，没有做视觉 diff | 需要基线截图 vs 当前截图的比对 |                                                                                                  
     | **CI/CD 集成没有路径** | 方案停在本地执行 | 需要 GitOps + 自动化报告推送 |                                                                                                                           
     | **session pin 始终是假设** | agent-browser 是否真的支持 --session | 需要验证或给出 fallback |                                                                                                        
                                                                                                                                                                                                            
     ---                                                                                                                                                                                                    
                                                                                                                                                                                                            
     ### 二、核心突破一：真并行架构（Multi-Browser Grid Orchestration）                                                                                                                                     
                                                                                                                                                                                                            
     #### 2.1 为什么 delegate_task 不是真正的并行                                                                                                                                                           
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
     delegate_task 的问题：                                                                                                                                                                                 
     ┌─────────────┐  ┌─────────────┐  ┌─────────────┐                                                                                                                                                      
     │  subagent 1 │  │  subagent 2 │  │  subagent 3 │                                                                                                                                                      
     │ (独立的browser│  │ (独立的browser│  │ (独立的browser│                                                                                                                                                
     │   session)   │  │   session)   │  │   session)   │                                                                                                                                                   
     └─────────────┘  └─────────────┘  └─────────────┘                                                                                                                                                      
     每个都要：打开浏览器 → 登录 → 导航 → 执行 → 关闭                                                                                                                                                       
     成本：登录 5s × N 个 agent = N 倍成本                                                                                                                                                                  
     且：共享的网络资源（同一个 HIP 网管）被 N 个浏览器同时操作 → 冲突                                                                                                                                      
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 2.2 正确方案：Hub-Spoke 模型                                                                                                                                                                      
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
                         ┌─────────────────┐                                                                                                                                                                
                         │  Orchestrator   │                                                                                                                                                                
                         │   (本agent)     │                                                                                                                                                                
                         │  负责调度 + 汇总  │                                                                                                                                                              
                         └────────┬────────┘                                                                                                                                                                
                                  │ 分发任务 & 收集结果                                                                                                                                                     
               ┌──────────────────┼──────────────────┐                                                                                                                                                      
               ↓                  ↓                  ↓                                                                                                                                                      
         ┌──────────┐       ┌──────────┐        ┌──────────┐                                                                                                                                                
         │ Grid Node│       │ Grid Node│        │ Grid Node│                                                                                                                                                
         │ :4441    │       │ :4442    │        │ :4443    │                                                                                                                                                
         │ (chrome1)│       │ (chrome2) │        │ (chrome3) │                                                                                                                                              
         └──────────┘       └──────────┘        └──────────┘                                                                                                                                                
     ```

     #### 2.3 Hub-Spoke 执行器实现                                                                                                                                                                          
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     #!/bin/bash                                                                                                                                                                                            
     # grid_executor.sh — 真并行执行器                                                                                                                                                                      
                                                                                                                                                                                                            
     GRID_HUB="http://localhost:4444"                                                                                                                                                                       
                                                                                                                                                                                                            
     # 通过 Selenium Grid API 下发任务                                                                                                                                                                      
     grid_execute() {                                                                                                                                                                                       
         local test_case="$1"                                                                                                                                                                               
         local capabilities="$2"  # e.g. "chrome,1920x1080" 或 "firefox"                                                                                                                                    
         local test_script="$3"                                                                                                                                                                             
         local session_id                                                                                                                                                                                   
                                                                                                                                                                                                            
         # 1. 向 Grid Hub 请求一个 session                                                                                                                                                                  
         session_id=$(curl -s -X POST "$GRID_HUB/session" \                                                                                                                                                 
             -H "Content-Type: application/json" \                                                                                                                                                          
             -d "{\"capabilities\":{\"browserName\":\"${capabilities%%,*}\"}}" \                                                                                                                            
             | python3 -c "import sys,json; print(json.load(sys.stdin)['value']['sessionId'])")                                                                                                             
                                                                                                                                                                                                            
         if [ -z "$session_id" ]; then                                                                                                                                                                      
             echo "[GRID] Failed to allocate node"                                                                                                                                                          
             return 1                                                                                                                                                                                       
         fi                                                                                                                                                                                                 
                                                                                                                                                                                                            
         echo "[GRID] Allocated session: $session_id"                                                                                                                                                       
                                                                                                                                                                                                            
         # 2. 通过 WebDriver Wire Protocol 执行测试                                                                                                                                                         
         # (agent-browser 底层也是走的 WebDriver 协议，可以 parallel 执行)                                                                                                                                  
                                                                                                                                                                                                            
         # 3. 上报结果                                                                                                                                                                                      
         curl -X DELETE "$GRID_HUB/session/$session_id"                                                                                                                                                     
                                                                                                                                                                                                            
         return 0                                                                                                                                                                                           
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 并行分发（使用 & + wait）                                                                                                                                                                            
     parallel_execute() {                                                                                                                                                                                   
         local tests=("$@")                                                                                                                                                                                 
         local pids=()                                                                                                                                                                                      
                                                                                                                                                                                                            
         for test in "${tests[@]}"; do                                                                                                                                                                      
             grid_execute "$test" "chrome" "test_script.sh" &                                                                                                                                               
             pids+=($!)                                                                                                                                                                                     
         done                                                                                                                                                                                               
                                                                                                                                                                                                            
         # 等待所有完成                                                                                                                                                                                     
         local failed=0                                                                                                                                                                                     
         for pid in "${pids[@]}"; do                                                                                                                                                                        
             wait $pid || failed=$((failed+1))                                                                                                                                                              
         done                                                                                                                                                                                               
                                                                                                                                                                                                            
         echo "[GRID] Parallel execution done: $failed failures"                                                                                                                                            
         return $failed                                                                                                                                                                                     
     }                                                                                                                                                                                                      
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 2.4 更实际的方案：多 Playwright 实例                                                                                                                                                              
                                                                                                                                                                                                            
     如果 agent-browser 底层是 Playwright，可以直接多实例并行：                                                                                                                                             
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     #!/usr/bin/env python3                                                                                                                                                                                 
     # multi_playwright_executor.py                                                                                                                                                                         
     from playwright.sync_api import sync_playwright                                                                                                                                                        
     import threading, time, json                                                                                                                                                                           
                                                                                                                                                                                                            
     class BrowserWorker(threading.Thread):                                                                                                                                                                 
         def __init__(self, worker_id, test_queue, results):                                                                                                                                                
             super().__init__()                                                                                                                                                                             
             self.worker_id = worker_id                                                                                                                                                                     
             self.test_queue = test_queue                                                                                                                                                                   
             self.results = results                                                                                                                                                                         
             self.playwright = None                                                                                                                                                                         
             self.browser = None                                                                                                                                                                            
                                                                                                                                                                                                            
         def run(self):                                                                                                                                                                                     
             self.playwright = sync_playwright().start()                                                                                                                                                    
             self.browser = self.playwright.chromium.launch(headless=False)                                                                                                                                 
                                                                                                                                                                                                            
             while True:                                                                                                                                                                                    
                 task = self.test_queue.get()                                                                                                                                                               
                 if task is None: break                                                                                                                                                                     
                 result = self.execute_task(task)                                                                                                                                                           
                 self.results.append(result)                                                                                                                                                                
                                                                                                                                                                                                            
             self.browser.close()                                                                                                                                                                           
             self.playwright.stop()                                                                                                                                                                         
                                                                                                                                                                                                            
         def execute_task(self, task):                                                                                                                                                                      
             context = self.browser.new_context(                                                                                                                                                            
                 viewport={'width': 1920, 'height': 1080},                                                                                                                                                  
                 ignore_https_errors=True                                                                                                                                                                   
             )                                                                                                                                                                                              
             page = context.new_page()                                                                                                                                                                      
                                                                                                                                                                                                            
             try:                                                                                                                                                                                           
                 page.goto(task['url'], timeout=30000)                                                                                                                                                      
                 page.wait_for_timeout(2000)                                                                                                                                                                
                 page.fill('input[name="username"]', task['username'])                                                                                                                                      
                 page.fill('input[name="password"]', task['password'])                                                                                                                                      
                 page.click('button[type="submit"]')                                                                                                                                                        
                 page.wait_for_timeout(3000)                                                                                                                                                                
                                                                                                                                                                                                            
                 return {'worker': self.worker_id, 'task': task['name'], 'status': 'PASS'}                                                                                                                  
             except Exception as e:                                                                                                                                                                         
                 return {'worker': self.worker_id, 'task': task['name'], 'status': 'FAIL', 'error': str(e)}                                                                                                 
             finally:                                                                                                                                                                                       
                 context.close()                                                                                                                                                                            
                                                                                                                                                                                                            
     # 主调度                                                                                                                                                                                               
     def run_parallel(tasks, concurrency=3):                                                                                                                                                                
         test_queue = queue.Queue()                                                                                                                                                                         
         for t in tasks: test_queue.put(t)                                                                                                                                                                  
         for _ in range(concurrency): test_queue.put(None)  # 哨兵                                                                                                                                          
                                                                                                                                                                                                            
         results = []                                                                                                                                                                                       
         workers = [BrowserWorker(i, test_queue, results) for i in range(concurrency)]                                                                                                                      
                                                                                                                                                                                                            
         for w in workers: w.start()                                                                                                                                                                        
         for w in workers: w.join()                                                                                                                                                                         
                                                                                                                                                                                                            
         return results                                                                                                                                                                                     
     ```

     ### 三、核心突破二：LLM-Driven 动态 Locator 生成                                                                                                                                                       
                                                                                                                                                                                                            
     #### 3.1 自愈的终极形态：没有"注册表"，只有 LLM                                                                                                                                                        
                                                                                                                                                                                                            
     当所有已知策略都失败时，人类会怎么做？                                                                                                                                                                 
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
     人：打开页面 → 看到按钮 → 知道在哪里 → 操作                                                                                                                                                            
     机器：没有注册表 → 不知道在哪里 → 失败                                                                                                                                                                 
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     LLM 做的是：**把"看到页面"这件事，用 vision 模型替代**。                                                                                                                                               
                                                                                                                                                                                                            
     #### 3.2 四层Locator生成器                                                                                                                                                                             
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
     ┌──────────────────────────────────────────────────────────┐                                                                                                                                           
     │              LLM Locator Generator (LLGL)                │                                                                                                                                           
     ├──────────────────────────────────────────────────────────┤                                                                                                                                           
     │ Layer 1: 注册表查询 (毫秒级, 已知元素)                    │                                                                                                                                          
     │   → return LOCATOR_REGISTRY[element_name]               │                                                                                                                                            
     ├──────────────────────────────────────────────────────────┤                                                                                                                                           
     │ Layer 2: 语义相似匹配 (次毫秒级, 见过但名字不同)          │                                                                                                                                          
     │   → NLP 语义向量匹配：element_name ≈ "新建业务" → "新建普通业务"│                                                                                                                                    
     │   → 用 cosine similarity 计算相似度 > 0.85 则直接用      │                                                                                                                                           
     ├──────────────────────────────────────────────────────────┤                                                                                                                                           
     │ Layer 3: Accessibility Tree 推理 (1-2秒, 没见过的元素)    │                                                                                                                                          
     │   → 把 snapshot 完整内容发给 LLM + prompt                │                                                                                                                                           
     │   → LLM 推断："源时隙编辑按钮" 对应 accessibility tree 中 │                                                                                                                                          
     │     的哪个 element，输出 ref 或 xpath                     │                                                                                                                                          
     ├──────────────────────────────────────────────────────────┤                                                                                                                                           
     │ Layer 4: Vision + 截图分析 (2-3秒, Accessibility 无效)   │                                                                                                                                           
     │   → 用 vision 截图发给 LLM：标注这个元素的位置           │                                                                                                                                           
     │   → 返回 (x, y) 坐标，用 mouse click                     │                                                                                                                                           
     └──────────────────────────────────────────────────────────┘                                                                                                                                           
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 3.3 Layer 3 实现：Accessibility Tree 推理                                                                                                                                                         
                                                                                                                                                                                                            
     ```python                                                                                                                                                                                              
     #!/usr/bin/env python3                                                                                                                                                                                 
     # llm_locator.py — LLM 驱动的动态 locator 生成                                                                                                                                                         
                                                                                                                                                                                                            
     import os, json, subprocess                                                                                                                                                                            
                                                                                                                                                                                                            
     LLM_PROVIDER = os.getenv("LLM_PROVIDER", "anthropic")  # anthropic / openai / local                                                                                                                    
                                                                                                                                                                                                            
     SYSTEM_PROMPT = """你是一个 UI 自动化工程师。你的任务：                                                                                                                                                
     给定一个 HIP 网管系统的 Accessibility Tree，以及要定位的元素描述，                                                                                                                                     
     输出该元素在 accessibility tree 中的确切位置。                                                                                                                                                         
                                                                                                                                                                                                            
     输出格式（JSON）：                                                                                                                                                                                     
     {                                                                                                                                                                                                      
         "strategy": "text | ref | xpath | aria-label | coordinate",                                                                                                                                        
         "value": "定位值",                                                                                                                                                                                 
         "confidence": 0.0-1.0,                                                                                                                                                                             
         "reasoning": "为什么这样定位"                                                                                                                                                                      
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     规则：                                                                                                                                                                                                 
     1. 优先用 text 定位（最稳定）                                                                                                                                                                          
     2. 其次用 aria-label / xpath                                                                                                                                                                           
     3. ref 数字不作为定位依据（动态变化）                                                                                                                                                                  
     4. 如果找不到，输出 confidence: 0.0                                                                                                                                                                    
     """                                                                                                                                                                                                    
                                                                                                                                                                                                            
     def get_accessibility_tree():                                                                                                                                                                          
         result = subprocess.run(                                                                                                                                                                           
             ['agent-browser', 'snapshot', '-c'],                                                                                                                                                           
             capture_output=True, text=True, timeout=30                                                                                                                                                     
         )                                                                                                                                                                                                  
         return result.stdout                                                                                                                                                                               
                                                                                                                                                                                                            
     def llm_query(tree_content, element_description):                                                                                                                                                      
         user_prompt = f"""Accessibility Tree:                                                                                                                                                              
     {tree_content[:8000]}  <!-- 截断避免 token 溢出 -->                                                                                                                                                    
                                                                                                                                                                                                            
     要定位的元素: {element_description}                                                                                                                                                                    
                                                                                                                                                                                                            
     请在上面的 Accessibility Tree 中找到该元素。                                                                                                                                                           
     """                                                                                                                                                                                                    
                                                                                                                                                                                                            
         if LLM_PROVIDER == "anthropic":                                                                                                                                                                    
             import anthropic                                                                                                                                                                               
             client = anthropic.Anthropic()                                                                                                                                                                 
             resp = client.messages.create(                                                                                                                                                                 
                 model="claude-sonnet-4-6-20250514",                                                                                                                                                        
                 max_tokens=512,                                                                                                                                                                            
                 system=SYSTEM_PROMPT,                                                                                                                                                                      
                 messages=[{"role": "user", "content": user_prompt}]                                                                                                                                        
             )                                                                                                                                                                                              
             return json.loads(resp.content[0].text)                                                                                                                                                        
                                                                                                                                                                                                            
         elif LLM_PROVIDER == "openai":                                                                                                                                                                     
             import openai                                                                                                                                                                                  
             resp = openai.chat.completions.create(                                                                                                                                                         
                 model="gpt-4o",                                                                                                                                                                            
                 messages=[                                                                                                                                                                                 
                     {"role": "system", "content": SYSTEM_PROMPT},                                                                                                                                          
                     {"role": "user", "content": user_prompt}                                                                                                                                               
                 ]                                                                                                                                                                                          
             )                                                                                                                                                                                              
             return json.loads(resp.choices[0].message.content)                                                                                                                                             
                                                                                                                                                                                                            
     def generate_locator(element_name, max_layers=4):                                                                                                                                                      
         # Layer 1: 注册表                                                                                                                                                                                  
         registry_result = resolve_from_registry(element_name)                                                                                                                                              
         if registry_result:                                                                                                                                                                                
             return registry_result, "registry"                                                                                                                                                             
                                                                                                                                                                                                            
         # Layer 2: 语义相似                                                                                                                                                                                
         semantic_result = resolve_from_semantic(element_name)                                                                                                                                              
         if semantic_result and semantic_result['confidence'] > 0.85:                                                                                                                                       
             return semantic_result, "semantic"                                                                                                                                                             
                                                                                                                                                                                                            
         # Layer 3: LLM 推理                                                                                                                                                                                
         tree = get_accessibility_tree()                                                                                                                                                                    
         llm_result = llm_query(tree, element_name)                                                                                                                                                         
         if llm_result['confidence'] > 0.7:                                                                                                                                                                 
             # 验证可用性                                                                                                                                                                                   
             if verify_locator(llm_result):                                                                                                                                                                 
                 return llm_result, "llm_tree"                                                                                                                                                              
             else:                                                                                                                                                                                          
                 # Layer 4: Vision fallback                                                                                                                                                                 
                 return generate_vision_locator(element_name), "llm_vision"                                                                                                                                 
                                                                                                                                                                                                            
         return None, "failed"                                                                                                                                                                              
                                                                                                                                                                                                            
     def verify_locator(locator):                                                                                                                                                                           
         """验证 LLM 生成的 locator 是否真的可用"""                                                                                                                                                         
         strategy = locator['strategy']                                                                                                                                                                     
         value = locator['value']                                                                                                                                                                           
                                                                                                                                                                                                            
         if strategy == "ref":                                                                                                                                                                              
             result = subprocess.run(                                                                                                                                                                       
                 ['agent-browser', 'snapshot', '-c'],                                                                                                                                                       
                 capture_output=True, text=True                                                                                                                                                             
             )                                                                                                                                                                                              
             return f"ref={value}" in result.stdout                                                                                                                                                         
                                                                                                                                                                                                            
         elif strategy == "text":                                                                                                                                                                           
             result = subprocess.run(                                                                                                                                                                       
                 ['agent-browser', 'snapshot', '-c'],                                                                                                                                                       
                 capture_output=True, text=True                                                                                                                                                             
             )                                                                                                                                                                                              
             return value in result.stdout                                                                                                                                                                  
                                                                                                                                                                                                            
         elif strategy == "coordinate":                                                                                                                                                                     
             x, y = value.split(',')                                                                                                                                                                        
             result = subprocess.run(                                                                                                                                                                       
                 ['agent-browser', 'mouse', 'click', x.strip(), y.strip()],                                                                                                                                 
                 capture_output=True, text=True                                                                                                                                                             
             )                                                                                                                                                                                              
             return result.returncode == 0                                                                                                                                                                  
                                                                                                                                                                                                            
         return False                                                                                                                                                                                       
     ```

     #### 3.5 自愈验证闭环（Self-Healing Verification Loop）                                                                                                                                                
                                                                                                                                                                                                            
     ```python                                                                                                                                                                                              
     def self_heal_with_verification(element_name, max_attempts=3):                                                                                                                                         
         """                                                                                                                                                                                                
         自愈标准流程：                                                                                                                                                                                     
         1. 尝试所有已知策略                                                                                                                                                                                
         2. 失败 → LLM 生成新策略                                                                                                                                                                           
         3. 验证新策略（执行一次 click，看是否报错）                                                                                                                                                        
         4. 验证成功 → 更新注册表 + 记录版本                                                                                                                                                                
         5. 验证失败 → 重试（最多3次）                                                                                                                                                                      
         """                                                                                                                                                                                                
                                                                                                                                                                                                            
         attempt = 0                                                                                                                                                                                        
         last_error = None                                                                                                                                                                                  
                                                                                                                                                                                                            
         while attempt < max_attempts:                                                                                                                                                                      
             attempt += 1                                                                                                                                                                                   
             locator, source = generate_locator(element_name)                                                                                                                                               
                                                                                                                                                                                                            
             if locator is None:                                                                                                                                                                            
                 print(f"[HEAL] All layers exhausted for '{element_name}'")                                                                                                                                 
                 return None, f"exhausted_after_{max_attempts}"                                                                                                                                             
                                                                                                                                                                                                            
             print(f"[HEAL] Attempt {attempt}: {source} → {locator['strategy']}={locator['value']}")                                                                                                        
                                                                                                                                                                                                            
             # 验证：执行这个 locator                                                                                                                                                                       
             success, error = verify_and_execute(locator)                                                                                                                                                   
                                                                                                                                                                                                            
             if success:                                                                                                                                                                                    
                 # 成功！学习这个新策略                                                                                                                                                                     
                 if source != "registry":                                                                                                                                                                   
                     update_registry(element_name, locator, source)                                                                                                                                         
                     save_registry_version()  # 写入 YAML，带时间戳版本                                                                                                                                     
                 print(f"[HEAL] SUCCESS: '{element_name}' now using {locator['strategy']}={locator['value']}")                                                                                              
                 return locator, source                                                                                                                                                                     
             else:                                                                                                                                                                                          
                 last_error = error                                                                                                                                                                         
                 # 标记此策略失败，下次不用                                                                                                                                                                 
                 blacklist_strategy(element_name, locator)                                                                                                                                                  
                                                                                                                                                                                                            
         return None, f"failed_after_{max_attempts}_attempts:{last_error}"                                                                                                                                  
     ```

     ### 四、核心突破三：视觉回归测试（Visual Regression Testing）                                                                                                                                          
                                                                                                                                                                                                            
     #### 4.1 什么时候需要视觉回归                                                                                                                                                                          
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
     文本类断言（grep 快照）够用：业务列表有 TestOTN01                                                                                                                                                      
     视觉类必须用截图比对：                                                                                                                                                                                 
       - 对话框弹出后，表单布局是否正确                                                                                                                                                                     
       - 下拉菜单展开后选项是否正确                                                                                                                                                                         
       - 按钮是否被截断、遮挡、重叠                                                                                                                                                                         
       - Java Applet 渲染的拓扑图是否正确                                                                                                                                                                   
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 4.2 实现：playwright screenshot + pixel diff                                                                                                                                                      
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     #!/bin/bash                                                                                                                                                                                            
     # visual_regression.sh                                                                                                                                                                                 
                                                                                                                                                                                                            
     BASELINE_DIR="./baselines/$(date +%Y%m%d)"                                                                                                                                                             
     CURRENT_DIR="./current/$(date +%Y%m%d)"                                                                                                                                                                
     DIFF_DIR="./diffs/$(date +%Y%m%d)"                                                                                                                                                                     
                                                                                                                                                                                                            
     mkdir -p "$BASELINE_DIR" "$CURRENT_DIR" "$DIFF_DIR"                                                                                                                                                    
                                                                                                                                                                                                            
     # 拍基准截图（首次运行或更新基线时）                                                                                                                                                                   
     capture_baseline() {                                                                                                                                                                                   
         local name="$1"                                                                                                                                                                                    
         agent-browser screenshot "$CURRENT_DIR/${name}.png"                                                                                                                                                
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 执行视觉 diff                                                                                                                                                                                        
     visual_diff() {                                                                                                                                                                                        
         local name="$1"                                                                                                                                                                                    
         local baseline="$BASELINE_DIR/${name}.png"                                                                                                                                                         
         local current="$CURRENT_DIR/${name}.png"                                                                                                                                                           
         local diff="$DIFF_DIR/${name}_diff.png"                                                                                                                                                            
                                                                                                                                                                                                            
         if [ ! -f "$baseline" ]; then                                                                                                                                                                      
             echo "[VISUAL] No baseline for '$name', saving as baseline"                                                                                                                                    
             cp "$current" "$baseline"                                                                                                                                                                      
             return 0                                                                                                                                                                                       
         fi                                                                                                                                                                                                 
                                                                                                                                                                                                            
         # 用 ImageMagick 做像素级 diff                                                                                                                                                                     
         convert "$baseline" "$current" \                                                                                                                                                                   
             -compose Difference \                                                                                                                                                                          
             -composite \                                                                                                                                                                                   
             -colorspace Gray \                                                                                                                                                                             
             "$diff"                                                                                                                                                                                        
                                                                                                                                                                                                            
         # 计算差异像素比例                                                                                                                                                                                 
         diff_ratio=$(identify -format "%[mean]" "$diff" 2>/dev/null | awk '{print $1/655.35}')                                                                                                             
                                                                                                                                                                                                            
         # 阈值：> 1% 差异则报告                                                                                                                                                                            
         threshold=1.0                                                                                                                                                                                      
         if (( $(echo "$diff_ratio > $threshold" | bc -l) )); then                                                                                                                                          
             echo "[VISUAL] FAIL: '$name' differs by ${diff_ratio}% (threshold: ${threshold}%)"                                                                                                             
             echo "[VISUAL] Diff saved: $diff"                                                                                                                                                              
             return 1                                                                                                                                                                                       
         else                                                                                                                                                                                               
             echo "[VISUAL] PASS: '$name' differs by ${diff_ratio}%"                                                                                                                                        
             return 0                                                                                                                                                                                       
         fi                                                                                                                                                                                                 
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 用法示例：对话框打开后截图比对                                                                                                                                                                       
     test_dialog_layout() {                                                                                                                                                                                 
         agent-browser click "新建普通业务"                                                                                                                                                                 
         agent-browser wait 2                                                                                                                                                                               
                                                                                                                                                                                                            
         # 关键截图点                                                                                                                                                                                       
         agent-browser screenshot "$CURRENT_DIR/dialog_form.png"                                                                                                                                            
         visual_diff "dialog_form"                                                                                                                                                                          
                                                                                                                                                                                                            
         # 填完表单后再截一次                                                                                                                                                                               
         agent-browser type "名称" "TestVisual"                                                                                                                                                             
         agent-browser screenshot "$CURRENT_DIR/dialog_filled.png"                                                                                                                                          
         visual_diff "dialog_filled"                                                                                                                                                                        
     }                                                                                                                                                                                                      
     ```

     ### 六、核心突破五：注册表自进化（Registry Self-Evolution）                                                                                                                                            
                                                                                                                                                                                                            
     #### 6.1 v2 自愈的核心缺陷：更新后不验证、无法回滚                                                                                                                                                     
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # v2 的问题：                                                                                                                                                                                          
     learn_from_success() {                                                                                                                                                                                 
         LOCATOR_REGISTRY[$name]="${working_strategy}|${old}"                                                                                                                                               
         save_registry  # 写入了，但从来不验证这次写入是否真的work                                                                                                                                          
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 如果写入后用例下次跑又失败：                                                                                                                                                                         
     # → 不知道是因为注册表更新引入了一个错误的新策略                                                                                                                                                       
     # → 还是因为其他原因                                                                                                                                                                                   
     # → 无法回滚到上一个正确版本                                                                                                                                                                           
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 6.2 解决方案：Git 式版本控制                                                                                                                                                                      
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # locator_versions.sh — 每个注册表更新都是一个 git commit                                                                                                                                              
                                                                                                                                                                                                            
     REGISTRY_DIR="./locator_registry"                                                                                                                                                                      
     mkdir -p "$REGISTRY_DIR"                                                                                                                                                                               
                                                                                                                                                                                                            
     # 快照当前注册表                                                                                                                                                                                       
     snapshot_registry() {                                                                                                                                                                                  
         local timestamp=$(date +%Y%m%d_%H%M%S)                                                                                                                                                             
         local snapshot_file="$REGISTRY_DIR/snapshot_${timestamp}.yaml"                                                                                                                                     
         # LOCATOR_REGISTRY 序列化到 YAML                                                                                                                                                                   
         declare -p LOCATOR_REGISTRY > "$snapshot_file"                                                                                                                                                     
         echo "[REGISTRY] Snapshot saved: $snapshot_file"                                                                                                                                                   
         echo "$snapshot_file"                                                                                                                                                                              
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 更新策略（新策略必须经过"影子测试"）                                                                                                                                                                 
     update_locator_with_rollback() {                                                                                                                                                                       
         local element_name="$1"                                                                                                                                                                            
         local new_locator="$2"                                                                                                                                                                             
         local source="$3"                                                                                                                                                                                  
                                                                                                                                                                                                            
         # 1. 先快照当前版本（回滚点）                                                                                                                                                                      
         local rollback_file=$(snapshot_registry)                                                                                                                                                           
                                                                                                                                                                                                            
         # 2. 执行影子测试：在当前页面试用新 locator，看是否有效                                                                                                                                            
         local test_result=$(eval_js "                                                                                                                                                                      
             (function() {                                                                                                                                                                                  
                 var el;                                                                                                                                                                                    
                 if ('${new_locator}' === 'text:${new_locator#text:}') {                                                                                                                                    
                     el = document.querySelector('[data-ref]');                                                                                                                                             
                 }                                                                                                                                                                                          
                 return el ? 'OK' : 'FAIL';                                                                                                                                                                 
             })()                                                                                                                                                                                           
         " 2>&1)                                                                                                                                                                                            
                                                                                                                                                                                                            
         if echo "$test_result" | grep -q "OK"; then                                                                                                                                                        
             # 3. 影子测试通过，提交更新                                                                                                                                                                    
             LOCATOR_REGISTRY[$element_name]="${new_locator}|${LOCATOR_REGISTRY[$element_name]}"                                                                                                            
             save_registry                                                                                                                                                                                  
                                                                                                                                                                                                            
             # 4. git commit 风格记录                                                                                                                                                                       
             echo "$(date +%s) $element_name $source -> $new_locator" >> "$REGISTRY_DIR/change_log.txt"                                                                                                     
             echo "[REGISTRY] Updated: $element_name = $new_locator (source: $source)"                                                                                                                      
         else                                                                                                                                                                                               
             # 5. 影子测试失败，恢复快照                                                                                                                                                                    
             cp "$rollback_file" "$REGISTRY_DIR/current.yaml"                                                                                                                                               
             echo "[REGISTRY] REJECTED: $new_locator failed shadow test for '$element_name'"                                                                                                                
             echo "[REGISTRY] Rolled back to: $rollback_file"                                                                                                                                               
         fi                                                                                                                                                                                                 
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 回滚到指定版本                                                                                                                                                                                       
     rollback_to() {                                                                                                                                                                                        
         local snapshot_file="$1"                                                                                                                                                                           
         if [ -f "$snapshot_file" ]; then                                                                                                                                                                   
             source "$snapshot_file"                                                                                                                                                                        
             save_registry                                                                                                                                                                                  
             echo "[REGISTRY] Rolled back to $snapshot_file"                                                                                                                                                
         else                                                                                                                                                                                               
             echo "[REGISTRY] ERROR: Snapshot not found: $snapshot_file"                                                                                                                                    
         fi                                                                                                                                                                                                 
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 列出可用版本                                                                                                                                                                                         
     list_versions() {                                                                                                                                                                                      
         echo "Available snapshots:"                                                                                                                                                                        
         ls -la "$REGISTRY_DIR"/snapshot_*.yaml | awk '{print $NF, $6, $7, $8}'                                                                                                                             
         echo ""                                                                                                                                                                                            
         echo "Change log:"                                                                                                                                                                                 
         cat "$REGISTRY_DIR/change_log.txt" | tail -20                                                                                                                                                      
     }                                                                                                                                                                                                      
     ```

     # 无 session 支持时，手动管理浏览器进程                                                                                                                                                                
     BROWSER_PID_FILE="/tmp/hip_browser.pid"

         echo $! > "$BROWSER_PID_FILE"                                                                                                                                                                      
         sleep 3                                                                                                                                                                                            
         echo "[BROWSER] Launched new browser (PID: $(cat $BROWSER_PID_FILE))"                                                                                                                              
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     cleanup_browser() {                                                                                                                                                                                    
         if [ -f "$BROWSER_PID_FILE" ]; then                                                                                                                                                                
             kill $(cat "$BROWSER_PID_FILE") 2>/dev/null                                                                                                                                                    
             rm -f "$BROWSER_PID_FILE"                                                                                                                                                                      
         fi                                                                                                                                                                                                 
     }
