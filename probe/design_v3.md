     ### 二、测试数据管理（Test Data Isolation）                                                                                                                                                            
                                                                                                                                                                                                            
     #### 2.1 问题本质                                                                                                                                                                                      
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 两个用例并行跑，都用了 TestOTN01                                                                                                                                                                     
     用例A: 创建业务 TestOTN01 → 用例B: 删除业务 TestOTN01                                                                                                                                                  
     # 结果：A 创建的业务被 B 意外删掉                                                                                                                                                                      
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 2.2 解决方案：时间戳 + 池化 + 预分配                                                                                                                                                              
                                                                                                                                                                                                            
     **方案 A：时间戳后缀（适合冒烟测试）**                                                                                                                                                                 
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 每次运行生成唯一标识，注入到所有测试数据                                                                                                                                                             
     RUN_ID="$(date +%s%N)"                                                                                                                                                                                 
     SERVICE_NAME="TestOTN_${RUN_ID: -6}"  # 取后6位：TestOTN_473821                                                                                                                                        
                                                                                                                                                                                                            
     # 优点：简单，绝对不冲突                                                                                                                                                                               
     # 缺点：无法跨用例关联（一个用例创建，另一个用例想删它，不知道名字）                                                                                                                                   
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     **方案 B：测试数据池（适合回归测试）**                                                                                                                                                                 
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # data_pool.sh — 集中管理测试数据的申请/释放                                                                                                                                                           
     declare -A DATA_POOL                                                                                                                                                                                   
                                                                                                                                                                                                            
     request_slot() {                                                                                                                                                                                       
         local slot="$1"  # 例如 "7-UO2X-OSE.V2"                                                                                                                                                            
         # 从池中找一个可用的 (slot未使用 且 timeslot未占用)                                                                                                                                                
         for entry in "${!DATA_POOL[@]}"; do                                                                                                                                                                
             if [[ "${DATA_POOL[$entry]}" == "FREE" ]] && [[ "$entry" == "$slot:"* ]]; then                                                                                                                 
                 DATA_POOL[$entry]="USED"                                                                                                                                                                   
                 echo "$entry"                                                                                                                                                                              
                 return 0                                                                                                                                                                                   
             fi                                                                                                                                                                                             
         done                                                                                                                                                                                               
         echo "ERROR: No free slot for $slot" >&2                                                                                                                                                           
         return 1                                                                                                                                                                                           
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     release_slot() {                                                                                                                                                                                       
         local key="$1"                                                                                                                                                                                     
         DATA_POOL[$key]="FREE"                                                                                                                                                                             
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 用法                                                                                                                                                                                                 
     SRC_INFO=$(request_slot "7-UO2X-OSE.V2")   # 返回 "7-UO2X-OSE.V2:IN/OUT:1:ODU2:1-ODU0:1"                                                                                                               
     DST_INFO=$(request_slot "8-UO8G-OSE.V2")                                                                                                                                                               
                                                                                                                                                                                                            
     # 解析：slot=${SRC_INFO%%:*}, rest=${SRC_INFO#*:}                                                                                                                                                      
     # 用例结束，不管成功失败，都 release                                                                                                                                                                   
     trap "release_slot '$SRC_INFO'; release_slot '$DST_INFO'" EXIT                                                                                                                                         
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     **方案 C：预分配 + 用例锁定（最适合 CI/CD）**                                                                                                                                                          
                                                                                                                                                                                                            
     ```bash

     ### 三、崩溃恢复机制（Crash Recovery）                                                                                                                                                                 
                                                                                                                                                                                                            
     #### 3.1 三层状态机                                                                                                                                                                                    
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
     用例执行中 → 崩溃(断电/网络中断/浏览器崩溃)                                                                                                                                                            
                         ↓                                                                                                                                                                                  
                 检测：进程消失？浏览器进程僵死？                                                                                                                                                           
                         ↓                                                                                                                                                                                  
                 恢复策略路由                                                                                                                                                                               
                 ↓          ↓          ↓                                                                                                                                                                    
             硬重启      重连会话    跳过                                                                                                                                                                   
             (最底层)    (中层)     (上层)                                                                                                                                                                  
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 3.2 实现：watchdog 守护进程                                                                                                                                                                       
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     #!/bin/bash                                                                                                                                                                                            
     # watchdog.sh — 用例执行守护，发现崩溃自动恢复                                                                                                                                                         
                                                                                                                                                                                                            
     execute_with_watchdog() {                                                                                                                                                                              
         local test_script="$1"                                                                                                                                                                             
         local timeout="${2:-300}"                                                                                                                                                                          
                                                                                                                                                                                                            
         # 启动测试脚本（后台）                                                                                                                                                                             
         $test_script &                                                                                                                                                                                     
         local pid=$!                                                                                                                                                                                       
                                                                                                                                                                                                            
         # 监控进程                                                                                                                                                                                         
         local elapsed=0                                                                                                                                                                                    
         while [ $elapsed -lt $timeout ]; do                                                                                                                                                                
             if ! kill -0 $pid 2>/dev/null; then                                                                                                                                                            
                 # 进程已退出，检查退出码                                                                                                                                                                   
                 wait $pid                                                                                                                                                                                  
                 local exit_code=$?                                                                                                                                                                         
                 if [ $exit_code -eq 0 ]; then                                                                                                                                                              
                     echo "[WATCHDOG] PASS: $test_script"                                                                                                                                                   
                 else                                                                                                                                                                                       
                     echo "[WATCHDOG] FAIL: $test_script (exit $exit_code)"                                                                                                                                 
                     # 清理脏数据                                                                                                                                                                           
                     cleanup_orphaned_resources                                                                                                                                                             
                 fi                                                                                                                                                                                         
                 return $exit_code                                                                                                                                                                          
             fi                                                                                                                                                                                             
                                                                                                                                                                                                            
             # 检测浏览器进程是否僵死（CPU 0% 持续 >60s）                                                                                                                                                   
             check_browser_stuck $pid                                                                                                                                                                       
             sleep 5                                                                                                                                                                                        
             elapsed=$((elapsed + 5))                                                                                                                                                                       
         done                                                                                                                                                                                               
                                                                                                                                                                                                            
         # 超时：杀掉进程，尝试恢复会话                                                                                                                                                                     
         echo "[WATCHDOG] TIMEOUT: killing $pid after ${timeout}s"                                                                                                                                          
         kill -9 $pid 2>/dev/null                                                                                                                                                                           
         recover_browser_session                                                                                                                                                                            
         return 124                                                                                                                                                                                         
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     check_browser_stuck() {                                                                                                                                                                                
         local pid=$1                                                                                                                                                                                       
         # ps 检测 CPU 使用率（需要 macOS/Linux ps 兼容）                                                                                                                                                   
         local cpu=$(ps -p $pid -o %cpu= 2>/dev/null | tr -d ' ')                                                                                                                                           
         if [[ $(echo "$cpu < 0.1" | bc -l) -eq 1 ]]; then                                                                                                                                                  
             stuck_count=$((stuck_count + 1))                                                                                                                                                               
             if [ $stuck_count -gt 12 ]; then  # 连续 60s CPU ~0                                                                                                                                            
                 echo "[WATCHDOG] Browser stuck, forcing recovery"                                                                                                                                          
                 kill -9 $pid                                                                                                                                                                               
                 recover_browser_session                                                                                                                                                                    
             fi                                                                                                                                                                                             
         else                                                                                                                                                                                               
             stuck_count=0                                                                                                                                                                                  
         fi                                                                                                                                                                                                 
     }                                                                                                                                                                                                      
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 3.3 脏数据清理（Orphan Cleanup）                                                                                                                                                                  
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     cleanup_orphaned_resources() {                                                                                                                                                                         
         # 重新连接浏览器会话                                                                                                                                                                               
         agent-browser attach --session orphaned                                                                                                                                                            
                                                                                                                                                                                                            
         # 列出所有业务，找 TestOTN_* 格式的                                                                                                                                                                
         agent-browser snapshot -c > /tmp/current_state.snap                                                                                                                                                
                                                                                                                                                                                                            
         grep -oP "TestOTN_\d{6}" /tmp/current_state.snap | sort -u | while read name; do                                                                                                                   
             local age=$(stat -f "%Sm" -t "%Y%m%d%H%M%S" /tmp/current_state.snap 2>/dev/null || echo "0")                                                                                                   
             local now=$(date +%Y%m%d%H%M%S)                                                                                                                                                                
             # 如果业务年龄 < 30分钟（说明是刚才崩溃的用例创建的），删除                                                                                                                                    
             if [ $((now - age)) -lt 30 ]; then                                                                                                                                                             
                 echo "[CLEANUP] Removing orphaned: $name"                                                                                                                                                  
                 agent-browser eval "/* 选中 $name 所在行 */"                                                                                                                                               
                 agent-browser click "删除"                                                                                                                                                                 
                 sleep 1                                                                                                                                                                                    
             fi                                                                                                                                                                                             
         done                                                                                                                                                                                               
     }                                                                                                                                                                                                      
     ```

     ### 四、eval 作用域缺陷的工程补偿                                                                                                                                                                      
                                                                                                                                                                                                            
     #### 4.1 问题根因                                                                                                                                                                                      
                                                                                                                                                                                                            
     agent-browser 的 `eval` 命令在执行时，**内部有变量声明复用机制**，导致 `let/const/var` 声明的标识符在连续调用间可能冲突：                                                                              
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
     第一次 eval: const x = 1; → 声明 x                                                                                                                                                                     
     第二次 eval: const x = 2; → ✗ SyntaxError: Identifier 'x' has already been declared                                                                                                                    
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     这在执行**批量操作序列**时是致命的。                                                                                                                                                                   
                                                                                                                                                                                                            
     #### 4.2 解决方案：每次 eval 用匿名函数封装 + eval 字符串拼接                                                                                                                                          
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 包装器：确保每次 eval 都是独立作用域                                                                                                                                                                 
     eval_js() {                                                                                                                                                                                            
         local js_code="$1"                                                                                                                                                                                 
         agent-browser eval "                                                                                                                                                                               
         (function() {                                                                                                                                                                                      
             $js_code                                                                                                                                                                                       
         })();                                                                                                                                                                                              
         " 2>&1                                                                                                                                                                                             
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 每次调用都包装成 IIFE，不污染外层作用域                                                                                                                                                              
     eval_js "const nodes = document.querySelectorAll('a'); console.log(nodes.length);"                                                                                                                     
     eval_js "const btns = document.querySelectorAll('button'); console.log(btns.length);"                                                                                                                  
     # 不会冲突！因为每次都是新的 IIFE scope                                                                                                                                                                
                                                                                                                                                                                                            
     # 对于复杂多步骤操作：用 ; 分隔，不要用 let 声明，用赋值                                                                                                                                               
     eval_js "                                                                                                                                                                                              
     var dlg = document.querySelector('[role=dialog]');                                                                                                                                                     
     if (!dlg) { console.log('NO_DIALOG'); }                                                                                                                                                                
     else {                                                                                                                                                                                                 
         var radios = dlg.querySelectorAll('input[type=radio]');                                                                                                                                            
         for (var i = 0; i < radios.length; i++) {                                                                                                                                                          
             if (radios[i].value && radios[i].value.indexOf('UO2X-OSE.V2 7') > -1) {                                                                                                                        
                 radios[i].click(); break;                                                                                                                                                                  
             }                                                                                                                                                                                              
         }                                                                                                                                                                                                  
     }                                                                                                                                                                                                      
     "                                                                                                                                                                                                      
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 4.3 批量多步操作的 eval 最佳实践                                                                                                                                                                  
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 对于 HIP 时隙选择对话框，一次 eval 搞定所有步骤（避免多次 call 带来的 ref 重分配问题）                                                                                                               
     eval_js "                                                                                                                                                                                              
     var MAX_WAIT = 5000, STEP = 300;                                                                                                                                                                       
     var found = false;                                                                                                                                                                                     
                                                                                                                                                                                                            
     function waitForDialog(cb) {                                                                                                                                                                           
         var dlgs = document.querySelectorAll('[role=dialog]');                                                                                                                                             
         for (var i = 0; i < dlgs.length; i++) {                                                                                                                                                            
             if (dlgs[i].textContent.includes('请选择时隙') && dlgs[i].offsetHeight > 0) {                                                                                                                  
                 cb(dlgs[i]); found = true; break;                                                                                                                                                          
             }                                                                                                                                                                                              
         }                                                                                                                                                                                                  
         if (!found) setTimeout(function() { waitForDialog(cb); }, STEP);                                                                                                                                   
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     waitForDialog(function(dlg) {                                                                                                                                                                          
         // 1. 选槽位7                                                                                                                                                                                      
         var radios = dlg.querySelectorAll('input[type=radio]');                                                                                                                                            
         for (var i = 0; i < radios.length; i++) {                                                                                                                                                          
             if (radios[i].value && radios[i].value.indexOf('UO2X-OSE.V2 7') > -1) {                                                                                                                        
                 radios[i].click(); break;                                                                                                                                                                  
             }                                                                                                                                                                                              
         }                                                                                                                                                                                                  
         // 等待端口渲染                                                                                                                                                                                    
         setTimeout(function() {                                                                                                                                                                            
             // 2. 选端口 IN/OUT:1                                                                                                                                                                          
             var ports = dlg.querySelectorAll('input[type=radio]');                                                                                                                                         
             for (var i = 0; i < ports.length; i++) {                                                                                                                                                       
                 if (ports[i].value === 'IN/OUT:1') { ports[i].click(); break; }                                                                                                                            
             }                                                                                                                                                                                              
             // 等待时隙checkbox渲染                                                                                                                                                                        
             setTimeout(function() {                                                                                                                                                                        
                 // 3. 选时隙                                                                                                                                                                               
                 var checks = dlg.querySelectorAll('input[type=checkbox]');                                                                                                                                 
                 for (var i = 0; i < checks.length; i++) {                                                                                                                                                  
                     if (checks[i].value && checks[i].value.indexOf('ODU2:1-ODU0:1') > -1) {                                                                                                                
                         checks[i].click(); break;                                                                                                                                                          
                     }                                                                                                                                                                                      
                 }                                                                                                                                                                                          
                 // 4. 点确定                                                                                                                                                                               
                 setTimeout(function() {                                                                                                                                                                    
                     var bts = dlg.querySelectorAll('button');                                                                                                                                              
                     for (var i = 0; i < bts.length; i++) {                                                                                                                                                 
                         if (bts[i].textContent.trim() === '确定') { bts[i].click(); break; }                                                                                                               
                     }                                                                                                                                                                                      
                     console.log('TIMESLOT_DIALOG_DONE');                                                                                                                                                   
                 }, STEP);                                                                                                                                                                                  
             }, STEP);                                                                                                                                                                                      
         }, STEP);                                                                                                                                                                                          
     });                                                                                                                                                                                                    
     " 2>&1                                                                                                                                                                                                 
     ```

     ### 五、错误分类与自动路由（Error Classification Engine）                                                                                                                                              
                                                                                                                                                                                                            
     #### 5.1 四类错误，路由不同                                                                                                                                                                            
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
     ┌─────────────────────────────────────────────────────────┐                                                                                                                                            
     │                     错误检测层                           │                                                                                                                                           
     │  agent-browser snapshot / eval 返回值分析                │                                                                                                                                           
     └────────────────────┬────────────────────────────────────┘                                                                                                                                            
                          ↓                                                                                                                                                                                 
              ┌───────────┼───────────┬────────────┐                                                                                                                                                        
              ↓           ↓           ↓            ↓                                                                                                                                                        
         [网络错误]   [业务错误]   [工具错误]   [环境错误]                                                                                                                                                  
         (重试3次)   (记录+跳过)  (降级方案)   (人工介入)                                                                                                                                                   
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 5.2 错误分类脚本                                                                                                                                                                                  
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     classify_error() {                                                                                                                                                                                     
         local output="$1"                                                                                                                                                                                  
                                                                                                                                                                                                            
         # 1. 网络/连接错误                                                                                                                                                                                 
         if echo "$output" | grep -qiE "ECONNREFUSED|ETIMEDOUT|ENOTFOUND|connection|timeout|network"; then                                                                                                  
             echo "NETWORK"                                                                                                                                                                                 
             return                                                                                                                                                                                         
         fi                                                                                                                                                                                                 
                                                                                                                                                                                                            
         # 2. 工具执行错误（ref不存在、eval语法错误）                                                                                                                                                       
         if echo "$output" | grep -qiE "ref.*not.*found|identifier.*declared|syntaxerror|cannot.*click|element.*missing"; then                                                                              
             echo "TOOL"                                                                                                                                                                                    
             return                                                                                                                                                                                         
         fi                                                                                                                                                                                                 
                                                                                                                                                                                                            
         # 3. 业务逻辑错误（权限不足、配置校验失败）                                                                                                                                                        
         if echo "$output" | grep -qiE "无权限|禁止|校验失败|失败|错误"; then                                                                                                                               
             echo "BUSINESS"                                                                                                                                                                                
             return                                                                                                                                                                                         
         fi                                                                                                                                                                                                 
                                                                                                                                                                                                            
         # 4. Session/State错误（浏览器僵死、对框状态异常）                                                                                                                                                 
         if echo "$output" | grep -qiE "session|stale|disconnected|browser.*dead"; then                                                                                                                     
             echo "SESSION"                                                                                                                                                                                 
             return                                                                                                                                                                                         
         fi                                                                                                                                                                                                 
                                                                                                                                                                                                            
         echo "UNKNOWN"                                                                                                                                                                                     
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 路由处理                                                                                                                                                                                             
     handle_error() {                                                                                                                                                                                       
         local error_type="$1"                                                                                                                                                                              
         local context="$2"                                                                                                                                                                                 
                                                                                                                                                                                                            
         case "$error_type" in                                                                                                                                                                              
             NETWORK)                                                                                                                                                                                       
                 echo "[RETRY] 网络错误，等待 5s 后重试..."                                                                                                                                                 
                 sleep 5                                                                                                                                                                                    
                 return 0  # 可重试                                                                                                                                                                         
                 ;;                                                                                                                                                                                         
             TOOL)                                                                                                                                                                                          
                 echo "[WARN] 工具错误，尝试降级方案..."                                                                                                                                                    
                 # 降级：用 vision 辅助定位                                                                                                                                                                 
                 agent-browser vision --question "图中 ${context} 按钮在哪里？" 2>&1                                                                                                                        
                 return 1  # 需人工检查                                                                                                                                                                     
                 ;;                                                                                                                                                                                         
             BUSINESS)                                                                                                                                                                                      
                 echo "[SKIP] 业务错误: $context"                                                                                                                                                           
                 return 2  # 跳过此用例                                                                                                                                                                     
                 ;;                                                                                                                                                                                         
             SESSION)                                                                                                                                                                                       
                 echo "[RECOVER] 会话错误，尝试恢复..."                                                                                                                                                     
                 recover_browser_session                                                                                                                                                                    
                 return 0  # 可重试                                                                                                                                                                         
                 ;;                                                                                                                                                                                         
             *)                                                                                                                                                                                             
                 echo "[FATAL] 未知错误: $output"                                                                                                                                                           
                 return 3  # 人工介入                                                                                                                                                                       
                 ;;                                                                                                                                                                                         
         esac                                                                                                                                                                                               
     }                                                                                                                                                                                                      
     ```

     ### 六、自愈机制闭环（Self-Healing Feedback Loop）                                                                                                                                                     
                                                                                                                                                                                                            
     #### 6.1 缺失的闭环                                                                                                                                                                                    
                                                                                                                                                                                                            
     原方案提到"自愈"，但**没有任何反馈回路**。真正的自愈需要：                                                                                                                                             
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
     失败发生 → 诊断(什么元素失效) → 学习(更新定位策略) → 验证(新策略有效) → 记录                                                                                                                           
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 6.2 实现：定位策略注册表 + 失败驱动更新                                                                                                                                                           
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # locator_registry.sh — 定位策略注册表                                                                                                                                                                 
     # 每个可定位元素有多个策略，按优先级尝试                                                                                                                                                               
                                                                                                                                                                                                            
     declare -A LOCATOR_REGISTRY                                                                                                                                                                            
                                                                                                                                                                                                            
     # 格式: LOCATOR["新建普通业务按钮"]="text:新建普通业务|aria-label:新建|class:.btn-primary|ref:e*                                                                                                       
                                                                                                                                                                                                            
     init_registry() {                                                                                                                                                                                      
         # 预置 HIP 系统常用元素的定位策略                                                                                                                                                                  
         LOCATOR_REGISTRY["新建普通业务按钮"]="text:新建普通业务|aria-label:新建普通业务|class:.btn-new"                                                                                                    
         LOCATOR_REGISTRY["OTN业务配置"]="text:OTN业务配置|ref:e191"                                                                                                                                        
         LOCATOR_REGISTRY["源时隙编辑按钮"]="class:.el-input[data-slot] button|aria-label:编辑源时隙"                                                                                                       
         LOCATOR_REGISTRY["时隙对话框确定"]="button:确定|role:dialog button.primary"                                                                                                                        
         LOCATOR_REGISTRY["业务名称输入框"]="placeholder:请输入|name:业务名称|class:.name-input"                                                                                                            
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 尝试解析定位器                                                                                                                                                                                       
     resolve_element() {                                                                                                                                                                                    
         local name="$1"                                                                                                                                                                                    
         local strategy="${LOCATOR_REGISTRY[$name]}"                                                                                                                                                        
         [ -z "$strategy" ] && { echo "ERROR: No strategy for $name"; return 1; }                                                                                                                           
                                                                                                                                                                                                            
         IFS='|' read -ra STRATS <<< "$strategy"                                                                                                                                                            
         for s in "${STRATS[@]}"; do                                                                                                                                                                        
             IFS=':' read -ra parts <<< "$s"                                                                                                                                                                
             local type="${parts[0]}"; local val="${parts[1]}"                                                                                                                                              
             case "$type" in                                                                                                                                                                                
                 text)   result=$(agent-browser snapshot -c 2>&1 | grep -m1 "$val");;                                                                                                                       
                 ref)    result=$(agent-browser snapshot -c 2>&1 | grep "ref=${val}");;                                                                                                                     
                 class)  result=$(agent-browser eval "document.querySelector('$val') ? 'FOUND' : 'MISSING'");;                                                                                              
                 aria-label) result=$(agent-browser eval "                                                                                                                                                  
                     (function(){ var e=document.querySelector('[aria-label*=\"$val\"]'); return e?'FOUND':'MISSING'; })()                                                                                  
                 ");;                                                                                                                                                                                       
             esac                                                                                                                                                                                           
             [ -n "$result" ] && { echo "${parts[0]}:${parts[1]}"; return 0; }                                                                                                                              
         done                                                                                                                                                                                               
         echo "ALL_STRATEGIES_FAILED"                                                                                                                                                                       
         return 1                                                                                                                                                                                           
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 失败后学习：记录成功的替代策略                                                                                                                                                                       
     learn_from_success() {                                                                                                                                                                                 
         local name="$1"                                                                                                                                                                                    
         local working_strategy="$2"  # e.g. "class:.btn-new"                                                                                                                                               
         local old="${LOCATOR_REGISTRY[$name]}"                                                                                                                                                             
         LOCATOR_REGISTRY[$name]="${working_strategy}|${old}"                                                                                                                                               
         echo "[HEAL] $name: added strategy '$working_strategy' (was: $old)"                                                                                                                                
         # 持久化到磁盘                                                                                                                                                                                     
         save_registry                                                                                                                                                                                      
     }                                                                                                                                                                                                      
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 6.3 自愈执行循环                                                                                                                                                                                  
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 执行带自愈的点击                                                                                                                                                                                     
     heal_click() {                                                                                                                                                                                         
         local element_name="$1"                                                                                                                                                                            
         local max_attempts="${2:-3}"                                                                                                                                                                       
                                                                                                                                                                                                            
         for attempt in $(seq 1 $max_attempts); do                                                                                                                                                          
             echo "[HEAL] Attempt $attempt for '$element_name'"                                                                                                                                             
                                                                                                                                                                                                            
             local strategy=$(resolve_element "$element_name")                                                                                                                                              
             if [ "$strategy" = "ALL_STRATEGIES_FAILED" ]; then                                                                                                                                             
                 echo "[HEAL] 所有定位策略失败，尝试快照全文扫描"                                                                                                                                           
                 agent-browser snapshot -c > /tmp/fail_snap_$(date +%s).snap                                                                                                                                
                 return 1                                                                                                                                                                                   
             fi                                                                                                                                                                                             
                                                                                                                                                                                                            
             echo "[HEAL] Using strategy: $strategy"                                                                                                                                                        
             IFS=':' read -ra parts <<< "$strategy"                                                                                                                                                         
             local type="${parts[0]}"; local val="${parts[1]}"                                                                                                                                              
                                                                                                                                                                                                            
             case "$type" in                                                                                                                                                                                
                 text)   agent-browser click "$val" && learn_from_success "$element_name" "$strategy" && return 0;;                                                                                         
                 ref)    agent-browser click "$val" && learn_from_success "$element_name" "$strategy" && return 0;;                                                                                         
                 class)  agent-browser eval "document.querySelector('$val').click(); console.log('CLICKED')" && return 0;;                                                                                  
             esac                                                                                                                                                                                           
                                                                                                                                                                                                            
             # 失败则下次用下一个策略                                                                                                                                                                       
             sleep 1                                                                                                                                                                                        
         done                                                                                                                                                                                               
                                                                                                                                                                                                            
         echo "[HEAL] Exhausted $max_attempts attempts for '$element_name'"                                                                                                                                 
         return 1                                                                                                                                                                                           
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 用法：替代直接 click                                                                                                                                                                                 
     heal_click "新建普通业务按钮"                                                                                                                                                                          
     heal_click "源时隙编辑按钮"                                                                                                                                                                            
     ```

     ### 七、测试框架化（Test Framework）                                                                                                                                                                   
                                                                                                                                                                                                            
     #### 7.1 脚本 vs 框架的本质差异                                                                                                                                                                        
                                                                                                                                                                                                            
     | 维度 | 纯脚本 | 轻量框架 |                                                                                                                                                                           
     |------|--------|----------|                                                                                                                                                                           
     | 共享 setup/teardown | 每个脚本自己写 | 框架自动调用 |                                                                                                                                                
     | 用例间数据隔离 | 靠约定 | 框架强制 |                                                                                                                                                                 
     | 失败后继续跑 | 手动判断 | `@continue_on_fail` 注解 |                                                                                                                                                 
     | 日志集中 | 散落各脚本 | 统一 report |                                                                                                                                                                
     | 覆盖率统计 | 无 | 框架自动统计 |                                                                                                                                                                     
                                                                                                                                                                                                            
     #### 7.2 最小框架实现（bash + YAML）                                                                                                                                                                   
                                                                                                                                                                                                            
     ```yaml

     ```bash                                                                                                                                                                                                
     #!/bin/bash                                                                                                                                                                                            
     # run_suite.sh — 轻量测试框架引擎                                                                                                                                                                      
                                                                                                                                                                                                            
     YAML_FILE="$1"                                                                                                                                                                                         
     TAG_FILTER="${2:-smoke}"  # 默认只跑 smoke                                                                                                                                                             
                                                                                                                                                                                                            
     parse_yaml() { ... }  # 简易 YAML parser                                                                                                                                                               
                                                                                                                                                                                                            
     load_suite() {                                                                                                                                                                                         
         local suite=$(parse_yaml "$YAML_FILE")                                                                                                                                                             
                                                                                                                                                                                                            
         # 执行 global_setup                                                                                                                                                                                
         echo "[FRAMEWORK] Running global setup..."                                                                                                                                                         
         eval "${suite.global_setup}"                                                                                                                                                                       
                                                                                                                                                                                                            
         # 按依赖顺序执行用例                                                                                                                                                                               
         local passed=0; local failed=0; local skipped=0                                                                                                                                                    
         for case in "${suite.cases[@]}"; do                                                                                                                                                                
             # Tag 过滤                                                                                                                                                                                     
             if [[ ! " ${case.tags[*]} " =~ " $TAG_FILTER " ]]; then                                                                                                                                        
                 echo "[FRAMEWORK] SKIP (tag): ${case.name}"                                                                                                                                                
                 skipped=$((skipped+1))                                                                                                                                                                     
                 continue                                                                                                                                                                                   
             fi                                                                                                                                                                                             
                                                                                                                                                                                                            
             # 依赖检查                                                                                                                                                                                     
             if [ -n "${case.depends}" ]; then                                                                                                                                                              
                 local dep_passed=true                                                                                                                                                                      
                 for dep in "${case.depends[@]}"; do                                                                                                                                                        
                     if [[ ! -f "/tmp/.pass_${dep}" ]]; then                                                                                                                                                
                         echo "[FRAMEWORK] SKIP (deps): ${case.name} needs ${dep}"                                                                                                                          
                         skipped=$((skipped+1))                                                                                                                                                             
                         dep_passed=false                                                                                                                                                                   
                         break                                                                                                                                                                              
                     fi                                                                                                                                                                                     
                 done                                                                                                                                                                                       
                 $dep_passed || continue                                                                                                                                                                    
             fi                                                                                                                                                                                             
                                                                                                                                                                                                            
             echo "[FRAMEWORK] RUN: ${case.name}"                                                                                                                                                           
                                                                                                                                                                                                            
             # 执行用例（超时控制）                                                                                                                                                                         
             timeout ${case.timeout} bash -c "                                                                                                                                                              
                 source ${case.name}.sh '${case.params}'                                                                                                                                                    
             " 2>&1                                                                                                                                                                                         
             local result=$?                                                                                                                                                                                
                                                                                                                                                                                                            
             if [ $result -eq 0 ]; then                                                                                                                                                                     
                 echo "[FRAMEWORK] PASS: ${case.name}"                                                                                                                                                      
                 touch "/tmp/.pass_${case.name}"                                                                                                                                                            
                 passed=$((passed+1))                                                                                                                                                                       
             elif [ $result -eq 124 ]; then                                                                                                                                                                 
                 echo "[FRAMEWORK] TIMEOUT: ${case.name}"                                                                                                                                                   
                 failed=$((failed+1))                                                                                                                                                                       
             else                                                                                                                                                                                           
                 if [ ${case.retry} -gt 0 ]; then                                                                                                                                                           
                     for r in $(seq 1 ${case.retry}); do                                                                                                                                                    
                         echo "[FRAMEWORK] RETRY $r: ${case.name}"                                                                                                                                          
                         bash -c "source ${case.name}.sh '${case.params}'" 2>&1                                                                                                                             
                         [ $? -eq 0 ] && break                                                                                                                                                              
                     done                                                                                                                                                                                   
                 fi                                                                                                                                                                                         
                 echo "[FRAMEWORK] FAIL: ${case.name}"                                                                                                                                                      
                 failed=$((failed+1))                                                                                                                                                                       
             fi                                                                                                                                                                                             
         done                                                                                                                                                                                               
                                                                                                                                                                                                            
         # 执行 global_teardown                                                                                                                                                                             
         echo "[FRAMEWORK] Running global teardown..."                                                                                                                                                      
         eval "${suite.global_teardown}"                                                                                                                                                                    
                                                                                                                                                                                                            
         echo "[FRAMEWORK] ===== SUMMARY ====="                                                                                                                                                             
         echo "[FRAMEWORK] PASSED: $passed"                                                                                                                                                                 
         echo "[FRAMEWORK] FAILED: $failed"                                                                                                                                                                 
         echo "[FRAMEWORK] SKIPPED: $skipped"                                                                                                                                                               
     }                                                                                                                                                                                                      
     ```

     ### 九、迭代要点总结                                                                                                                                                                                   
                                                                                                                                                                                                            
     | 版本 | 核心贡献 | 关键缺陷 |                                                                                                                                                                         
     |------|----------|----------|                                                                                                                                                                         
     | v1 初稿 | 方向性框架（分层/等待/复用） | 过度设计、重侦察轻执行、无闭环 |                                                                                                                            
     | v1 反思 | 指出 ref ID 脆弱、进程开销大问题 | 只破不立，未给具体解法 |                                                                                                                                
     | v2 本篇 | eval 作用域补偿、错误分类路由、自愈闭环、轻量框架 | 数据池方案仍偏简单，未考虑分布式并行 |                                                                                                 
                                                                                                                                                                                                            
     **v3 应重点攻克**：分布式多浏览器并行执行框架（当前只讨论了单浏览器会话内的伪并行）、以及 LLM-based 动态定位策略生成（当所有已知策略都失败时，让 LLM 根据截图/树理解页面结构，动态生成新               
     locator）。
