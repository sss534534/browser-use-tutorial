## agent-browser 自动化测试 — 方案设计                                                                                                                                                                 
                                                                                                                                                                                                            
     ### 一、前置侦察阶段（Pre-test Reconnaissance）                                                                                                                                                        
                                                                                                                                                                                                            
     #### 1.1 页面结构快照存档                                                                                                                                                                              
     每次进入新功能模块前，先做一次完整快照，作为"基准地图"保存。                                                                                                                                           
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 入口处存档：模块打开前的状态                                                                                                                                                                         
     agent-browser snapshot -c > baseline_<module>_<timestamp>.snap                                                                                                                                         
                                                                                                                                                                                                            
     # 对话框打开前存档：记录 ref ID 分布                                                                                                                                                                   
     agent-browser snapshot -c > dialog_<name>_before.snap                                                                                                                                                  
                                                                                                                                                                                                            
     # 关键 ref ID 提取为结构化数据                                                                                                                                                                         
     agent-browser snapshot -c | grep -E "button|textbox|radio|checkbox" > refs_<module>.txt                                                                                                                
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     **目的**：当 ref ID 随页面渲染动态变化时，可通过 `snapshot -c` 重新获取，同时通过对比两次快照快速定位新增/消失的元素，避免依赖"记忆中的旧 ref"导致点击失败。                                           
                                                                                                                                                                                                            
     #### 1.2 JavaScript 上下文探测                                                                                                                                                                         
     在复杂 SPA/Java Applet 页面中，很多状态藏在 DOM 深处，光靠 accessibility tree 看不到。                                                                                                                 
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 探测隐藏的 JS 全局函数（导航、弹窗、树操作）                                                                                                                                                         
     agent-browser eval "                                                                                                                                                                                   
     Object.keys(window).filter(k =>                                                                                                                                                                        
       typeof window[k] === 'function' &&                                                                                                                                                                   
       /menu|click|navigate|open|enter|select|find/i.test(k)                                                                                                                                                
     ).slice(0, 50).forEach(k => console.log(k, window[k].toString().slice(0, 80)))                                                                                                                         
     " 2>&1                                                                                                                                                                                                 
                                                                                                                                                                                                            
     # 探测 ztree/jstree 等树组件的内部 API                                                                                                                                                                 
     agent-browser eval "                                                                                                                                                                                   
     const zTree = $.fn.zTree.getZTreeObj('topoTree');                                                                                                                                                      
     if (zTree) {                                                                                                                                                                                           
       console.log('nodes:', zTree.getNodes().length);                                                                                                                                                      
       zTree.getNodes().forEach(n => console.log(n.name, n.tId, n.id));                                                                                                                                     
     }                                                                                                                                                                                                      
     " 2>&1                                                                                                                                                                                                 
                                                                                                                                                                                                            
     # 探测弹窗/对话框是否存在（即便 accessibility tree 没显示）                                                                                                                                            
     agent-browser eval "                                                                                                                                                                                   
     document.querySelectorAll('[role=dialog], .el-dialog, .layui-layer').forEach(d => {                                                                                                                    
       if (d.offsetHeight > 0) console.log('visible dialog:', d.className, d.textContent.slice(0, 50));                                                                                                     
     });                                                                                                                                                                                                    
     " 2>&1                                                                                                                                                                                                 
                                                                                                                                                                                                            
     # 探测当前激活的 iframe（Java Applet 通常藏在 iframe 里）                                                                                                                                              
     agent-browser eval "                                                                                                                                                                                   
     document.querySelectorAll('iframe').forEach(f => {                                                                                                                                                     
       try { console.log(f.id, f.name, f.src); } catch(e) { console.log(f.id, 'blocked'); }                                                                                                                 
     });                                                                                                                                                                                                    
     " 2>&1                                                                                                                                                                                                 
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     **目的**：提前发现 `contextMenu(event)`、`enterNE(nodeId)`、`getSelectedNode()` 这类隐藏 API，避免盲目模拟鼠标事件。 HIP 系统中这些函数是自动化成功的关键。

     ### 二、用例结构化设计（Test Case Architecture）                                                                                                                                                       
                                                                                                                                                                                                            
     #### 2.1 黄金断言链（Golden Assertion Chain）                                                                                                                                                          
     每个用例按 **GIVEN / WHEN / THEN / CLEANUP** 四段式编写，cleanup 阶段确保环境影响不泄漏到下一个用例：                                                                                                  
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # ===== GIVEN: 前置状态准备 =====                                                                                                                                                                      
     # 登录 → 进入网元 → 打开功能页面 → 打开对话框                                                                                                                                                          
                                                                                                                                                                                                            
     # ===== WHEN: 核心操作 =====                                                                                                                                                                           
     # 填写表单 → 选择下拉 → 点击确定                                                                                                                                                                       
                                                                                                                                                                                                            
     # ===== THEN: 结果断言 =====                                                                                                                                                                           
     # agent-browser snapshot -c | grep "预期值"                                                                                                                                                            
     # assert "激活" in snapshot                                                                                                                                                                            
     # assert "Up" in snapshot                                                                                                                                                                              
                                                                                                                                                                                                            
     # ===== CLEANUP: 环境恢复 =====                                                                                                                                                                        
     # 关闭对话框 / 删除刚创建的测试数据 / 返回上一页                                                                                                                                                       
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 2.2 用例分层：原子操作 → 组合操作 → 业务场景                                                                                                                                                      
                                                                                                                                                                                                            
     ```                                                                                                                                                                                                    
     Layer 1 - 原子操作（最细粒度，可跨用例复用）                                                                                                                                                           
       click_tree_node(nodeName)          # 点击功能树节点                                                                                                                                                  
       open_dialog(dialogName)            # 打开指定对话框                                                                                                                                                  
       select_radio_with_label(label)     # 选择指定标签的单选                                                                                                                                              
       select_checkbox(label)             # 选择复选框                                                                                                                                                      
                                                                                                                                                                                                            
     Layer 2 - 组合操作（特定模块内的标准流程）                                                                                                                                                             
       otn_enter_ne_manager(neName)        # 进入网元管理器                                                                                                                                                 
       otn_open_new_service_dialog()       # 打开新建业务对话框                                                                                                                                             
       otn_select_source_timeslot(slot, port, odu)  # 选择源时隙                                                                                                                                            
                                                                                                                                                                                                            
     Layer 3 - 业务场景（端到端测试用例）                                                                                                                                                                   
       test_create_otn_normal_service()    # 创建普通OTN业务                                                                                                                                                
       test_create_otn_protected_service() # 创建保护OTN业务                                                                                                                                                
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 2.3 参数化用例模板                                                                                                                                                                                
     将变化的部分提取为变量，避免用例膨胀：                                                                                                                                                                 
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 模板：create_otn_service.sh                                                                                                                                                                          
     SERVICE_NAME="${1:-TestOTN_$(date +%s)}"                                                                                                                                                               
     SRC_SLOT="${2:-7}"                                                                                                                                                                                     
     DST_SLOT="${3:-8}"                                                                                                                                                                                     
     SRC_PORT="${4:-IN/OUT:1}"                                                                                                                                                                              
     DST_PORT="${5:-IN/OUT:1}"                                                                                                                                                                              
                                                                                                                                                                                                            
     # 固定流程...                                                                                                                                                                                          
     agent-browser type e138 "$SERVICE_NAME"                                                                                                                                                                
     agent-browser click e140  # 源时隙编辑                                                                                                                                                                 
     # ... 参数化部分 ...                                                                                                                                                                                   
     agent-browser click e151  # 确定                                                                                                                                                                       
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 重复执行：每次用不同参数                                                                                                                                                                             
     ./create_otn_service.sh TestOTN_A 7 8 "IN/OUT:1" "IN/OUT:1"                                                                                                                                            
     ./create_otn_service.sh TestOTN_B 7 8 "IN/OUT:2" "IN/OUT:2"                                                                                                                                            
     ./create_otn_service.sh TestOTN_C 11 13 "IN/OUT:1" "IN/OUT:1"                                                                                                                                          
     ```

     ### 三、导航序列复用（Navigation Reuse）                                                                                                                                                               
                                                                                                                                                                                                            
     #### 3.1 可复用的导航脚本库                                                                                                                                                                            
     将 HIP 系统的标准导航路径脚本化：                                                                                                                                                                      
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # login_and_enter_ne.sh                                                                                                                                                                                
     # 用法: ./login_and_enter_ne.sh <url> <user> <pass> <ne_ip>                                                                                                                                            
     URL="$1"; USER="$2"; PASS="$3"; NE_IP="$4"                                                                                                                                                             
                                                                                                                                                                                                            
     agent-browser open "$URL" --headed                                                                                                                                                                     
     agent-browser wait 2                                                                                                                                                                                   
     agent-browser type e1 "$USER"                                                                                                                                                                          
     agent-browser type e2 "$PASS"                                                                                                                                                                          
     agent-browser click login_btn                                                                                                                                                                          
     agent-browser wait 3                                                                                                                                                                                   
                                                                                                                                                                                                            
     # 主拓扑 → 展开树 → 右键进入网元                                                                                                                                                                       
     agent-browser click main_topology_menu                                                                                                                                                                 
     agent-browser wait 2                                                                                                                                                                                   
     agent-browser eval "/* 右键触发全部展开 */"                                                                                                                                                            
     agent-browser click "全部展开"                                                                                                                                                                         
     agent-browser wait 1                                                                                                                                                                                   
     agent-browser eval "/* 定位 $NE_IP 节点并右键 */"                                                                                                                                                      
     agent-browser click "网元管理器"                                                                                                                                                                       
     agent-browser wait 2                                                                                                                                                                                   
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 3.2 分层等待策略                                                                                                                                                                                  
     不要用固定 `sleep N`，改用"条件等待"减少不必要的延时：                                                                                                                                                 
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 差：固定等待                                                                                                                                                                                         
     sleep 5                                                                                                                                                                                                
                                                                                                                                                                                                            
     # 好：轮询等待目标出现（超时10秒）                                                                                                                                                                     
     wait_for() {                                                                                                                                                                                           
       local target="$1"                                                                                                                                                                                    
       local timeout="${2:-10}"                                                                                                                                                                             
       local elapsed=0                                                                                                                                                                                      
       while [ $elapsed -lt $timeout ]; do                                                                                                                                                                  
         if agent-browser snapshot -c 2>&1 | grep -q "$target"; then                                                                                                                                        
           echo "Found '$target' after ${elapsed}s"                                                                                                                                                         
           return 0                                                                                                                                                                                         
         fi                                                                                                                                                                                                 
         sleep 1                                                                                                                                                                                            
         elapsed=$((elapsed + 1))                                                                                                                                                                           
       done                                                                                                                                                                                                 
       echo "Timeout waiting for '$target' after ${timeout}s"                                                                                                                                               
       return 1                                                                                                                                                                                             
     }                                                                                                                                                                                                      
                                                                                                                                                                                                            
     # 用法                                                                                                                                                                                                 
     wait_for "OTN业务配置"                                                                                                                                                                                 
     wait_for "新建普通业务"                                                                                                                                                                                
     wait_for "TestOTN01"  # 创建后验证                                                                                                                                                                     
     ```

     ### 四、对话状态感知（Dialog State Management）                                                                                                                                                        
                                                                                                                                                                                                            
     #### 4.1 对话框自动探测与关闭                                                                                                                                                                          
     在 HIP 系统中，对话框可能在多个 ref 下重复出现或嵌套，用 JS 统一处理：                                                                                                                                 
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # auto_close_dialog.sh — 关闭任意弹窗                                                                                                                                                                  
     close_any_dialog() {                                                                                                                                                                                   
       agent-browser eval "                                                                                                                                                                                 
       // 遍历所有可见弹窗并关闭                                                                                                                                                                            
       const selectors = ['[role=dialog]', '.el-dialog', '.layui-layer', '[aria-label]'];                                                                                                                   
       selectors.forEach(sel => {                                                                                                                                                                           
         document.querySelectorAll(sel).forEach(d => {                                                                                                                                                      
           if (d.offsetHeight > 0 && d.textContent.includes('请选择')) {                                                                                                                                    
             const btns = d.querySelectorAll('button');                                                                                                                                                     
             for (const b of btns) {                                                                                                                                                                        
               if (b.textContent.trim() === '确定') { b.click(); console.log('closed'); }                                                                                                                   
             }                                                                                                                                                                                              
           }                                                                                                                                                                                                
         });                                                                                                                                                                                                
       });                                                                                                                                                                                                  
       " 2>&1                                                                                                                                                                                               
       sleep 1                                                                                                                                                                                              
     }                                                                                                                                                                                                      
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 4.2 残留表单状态清理                                                                                                                                                                              
     HIP 系统中，新建业务对话框关闭后再次打开，残留值仍在。使用 `click 功能树节点` 刷新页面状态：                                                                                                           
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     cleanup_dialog() {                                                                                                                                                                                     
       # 点击功能树节点重置表单（已知技巧）                                                                                                                                                                 
       agent-browser click "OTN业务配置"                                                                                                                                                                    
       agent-browser wait 2                                                                                                                                                                                 
       agent-browser click "新建普通业务"                                                                                                                                                                   
       agent-browser wait 2                                                                                                                                                                                 
     }                                                                                                                                                                                                      
     ```

     ### 五、知识固化（Knowledge Persistence）                                                                                                                                                              
                                                                                                                                                                                                            
     #### 5.1 Ref ID 映射表维护                                                                                                                                                                             
     在 skills 中维护结构化映射，每次发现新的 ref 规律就更新：                                                                                                                                              
                                                                                                                                                                                                            
     ```yaml                                                                                                                                                                                                
     # SKILL.md 或单独的 refs.yaml                                                                                                                                                                          
     otn_new_service_dialog:                                                                                                                                                                                
       描述: 新建普通业务对话框                                                                                                                                                                             
       字段:                                                                                                                                                                                                
         名称输入框: { ref: e138, type: textbox }                                                                                                                                                           
         源板位: { ref: e170, type: textbox, 缺省值: "7-UO2X-OSE.V2" }                                                                                                                                      
         宿板位: { ref: e172, type: textbox, 缺省值: "8-UO8G-OSE.V2" }                                                                                                                                      
         源时隙编辑按钮: { ref: e140 }                                                                                                                                                                      
         宿时隙编辑按钮: { ref: e141 }                                                                                                                                                                      
         确定按钮: { ref: e151 }                                                                                                                                                                            
         取消按钮: { ref: e152 }                                                                                                                                                                            
       时隙选择对话框:                                                                                                                                                                                      
         槽位7单选: { ref: e79 }                                                                                                                                                                            
         端口IN/OUT1: { ref: e95 }                                                                                                                                                                          
         ODU2-ODU0-1复选: { ref: e46 }                                                                                                                                                                      
         确定按钮: { ref: e14 }                                                                                                                                                                             
       注意事项:                                                                                                                                                                                            
         - type命令对自定义组件无效，需用click+eval组合                                                                                                                                                     
         - 对话框确定按钮需用JS定位（ref不稳定）                                                                                                                                                            
         - 表单重置需点击功能树节点                                                                                                                                                                         
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 5.2 失败模式知识库                                                                                                                                                                                
     每次失败后，记录"什么操作 + 什么错误 + 如何恢复"：                                                                                                                                                     
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # failure_log.md（追加到每次测试报告）                                                                                                                                                                 
     ## 2026-04-22 15:47                                                                                                                                                                                    
     - 操作: agent-browser click e252 (时隙对话框确定)                                                                                                                                                      
     - 期望: 关闭对话框，填入时隙值                                                                                                                                                                         
     - 实际: 对话框保持开放，e252是外层确定不是对话框内确定                                                                                                                                                 
     - 根因: snapshot中e252是外层表单的确定，不是"请选择时隙"对话框内的                                                                                                                                     
     - 修复: 用JS遍历找到包含"请选择时隙"的dialog，再找其内部的确定按钮                                                                                                                                     
     - 恢复: agent-browser eval "...JS click..."                                                                                                                                                            
     ```

     ### 六、执行效率提升                                                                                                                                                                                   
                                                                                                                                                                                                            
     #### 6.1 用例并行化（适合多网元/多参数场景）                                                                                                                                                           
     用 `delegate_task` 并行跑多个独立用例：                                                                                                                                                                
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 并行测试3个不同网元的同一业务创建                                                                                                                                                                    
     delegate_task tasks='[                                                                                                                                                                                 
       {"goal": "在192.192.18.177上创建OTN业务TestA", "context": "NE=177, service=TestA"},                                                                                                                  
       {"goal": "在192.192.18.178上创建OTN业务TestB", "context": "NE=178, service=TestB"},                                                                                                                  
       {"goal": "在192.192.18.179上创建OTN业务TestC", "context": "NE=179, service=TestC"}                                                                                                                   
     ]'                                                                                                                                                                                                     
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 6.2 快速路径 vs 完整路径                                                                                                                                                                          
     区分"冒烟测试"（快速路径）和"完整回归"（完整路径）：                                                                                                                                                   
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     # 快速冒烟：只验证核心创建流程，跳过可选参数                                                                                                                                                           
     agent-browser click e42  # 新建普通业务                                                                                                                                                                
     agent-browser type e138 "SmokeTest"                                                                                                                                                                    
     agent-browser click e140; sleep 2; agent-browser click e79; sleep 1                                                                                                                                    
     agent-browser click e95; sleep 1; agent-browser click e46; sleep 1                                                                                                                                     
     agent-browser eval "/* click 确定 */"; sleep 1                                                                                                                                                         
     agent-browser click e141; sleep 2; agent-browser click e29; sleep 1                                                                                                                                    
     agent-browser click e71; sleep 1; agent-browser click e40; sleep 1                                                                                                                                     
     agent-browser eval "/* click 确定 */"; sleep 2                                                                                                                                                         
     agent-browser click e151; sleep 3                                                                                                                                                                      
     # 总计约 20s                                                                                                                                                                                           
                                                                                                                                                                                                            
     # 完整回归：每个字段都填+验证，总计约 3-5 分钟                                                                                                                                                         
     ```                                                                                                                                                                                                    
                                                                                                                                                                                                            
     #### 6.3 agent-browser 命令批量执行                                                                                                                                                                    
     用 heredoc 减少进程启动开销：                                                                                                                                                                          
                                                                                                                                                                                                            
     ```bash                                                                                                                                                                                                
     agent-browser << 'EOF'                                                                                                                                                                                 
     snapshot -c > before.snap                                                                                                                                                                              
     click e42                                                                                                                                                                                              
     wait 2                                                                                                                                                                                                 
     snapshot -c > after_dialog.snap                                                                                                                                                                        
     type e138 "RegressionTest"                                                                                                                                                                             
     EOF                                                                                                                                                                                                    
     ```
